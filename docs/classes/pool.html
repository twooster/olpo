    • Defined in pool.ts:391

    Acquires an item from the pool. Calls the provided callback with the item upon acquisition, and automatically releases it back to the pool after the callback returns (or resolves if it returns a promise).

    There are two ways to call this method:

    • If opts.wrappedItem is not set, or is false, then the unwrapped pool item (e.g., of type-param T) will be passed to the callback.
    • If opts.wrappedItem is true, then instead the callback will receive the "wrapped" pool Item, so you can inspect it.

    Parameters

      if true, indicates that all waiting acquisition promises should be rejected with a PoolDisposingError; otherwise these pool items will be allowed to resolve as normal
